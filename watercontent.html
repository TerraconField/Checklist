Water Content and Unit Weight (Density) Determination

  • For this lab entry, select Input > Lab Testing > Wc Density.
  • Select your boring number at the top.
  • Drag and align gINT columns to fit the entries from the lab sheets.
  • Follow the red numberings (1-6) and enter data for each required fields.
  • Wt Tare (g) is 45g for each number of rings(or Sample Height 1 (in)) present. In our example we have 6, so 45*6 = 270. If we had 5 rings, then our input would have been 225.
  • Diameter (mm) is a constant of 61.37 for the type of rings that we use. Populate the same number down the column for each lab entry.
  • Wet Density, Water Content (%), Dry Density, and Height (mm) will automatically calculate when you hit save.
